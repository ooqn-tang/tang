# 介绍
系统进行大规模变更，

模板应用程序 转为 单页应用程序

单页应用程序使用特色方法实现SEO优化

### 开发
执行 tang-generator GeneratorApplication mybatis-generator
执行 tang-gateway TangApplication main方法启动

然后就可以curd了

#### 软件架构
`SpringBoot2.3 + MyBatis + SpringSecurity5 + thymeleaf + Bootstrap3 + MySQL + Vue3 + Vite3`
